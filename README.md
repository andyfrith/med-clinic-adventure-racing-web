# med-clinic-adventure-racing-web

## Input from stakeholders
- Address a set of features addressing racers and their aiments, injuries, immediate health concerns, the clinic, doctors, health care workers providing care to racers and the booking, management or appointments or sessions to address the racer's medical attention needs.
- Need a dashboard application build upon a reliable, robust, well-tested and audited architecture created with Typescript, Next.js, React, Postgres, etc.
- Site should be attractive, great UX, is responsive, works well in a modern browser.
